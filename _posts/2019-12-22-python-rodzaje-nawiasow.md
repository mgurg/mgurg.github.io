### Nawiasy okrągłe `(...)` 

Funkcje `print('Hello world!')` czy tuple `(1, 2, 3)`

Definiowanie tuple: 
```python
`my_tuple = (0, 1, 2)`
```
### Nawiasy kwadratowe `[...]` 

Listy `[1, 2, 3]`, dostęp do poszczególnych elementów w list czy tuple: `my_list[0]` lub `my_tuple[0]`

Definiowanie list:
```python
`my_list= [4, 5, 5, 3, 4]`
```

### Nawiasy klamrowe `{...}` 

Definiowanie słowników: `{'key': 'value'}`

Definiowanie słownika:

```python
my_dict = {
    'Jan': 4,
    'Adam': 5,
    'Ania': 5,
    'Adrian': 3,
    'Agnieszka': 4
}
```
