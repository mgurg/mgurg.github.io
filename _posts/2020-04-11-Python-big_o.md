# Złożoność obliczeniowa

*Big O notation*:

O(1) — złożoność stała
O(logn) — logarytmiczna
O(n) — liniowa
O(nlogn) — liniowo-logarytmiczna
O(n²) — kwadratowa
...
O(n!)
